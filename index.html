SoftQuote

SoftQuote is a program built to handle the entire quote process from creation to invoice. There are three user levels staff, admin, and customer. Quotes are composed of date and time needed, tax rate, term, customer, customer location, customer employee, and products. Admin have the ability to create, update and delete users, quotes, tax rates, customers, customer employees, customer locations, terms, and products. The program has an automated pre approval process, automated emailing, automated pdf creation and saving and allows for customer revisions.

OperationDialOut

OperationDialOut is a program built to primarily handle automated outbound dialing. The program also allows for inbound calling and voicemails. There are three user levels: agents, admin, and superUser. The superUser interface allows for uploading and naming lists of customer information. From these lists superUsers can create and name calling campaigns from three different types. The superUsers can start, stop, and delete campaigns. The campaigns are automated and iterate through customers to exhaustion, calling customers and once answered connecting customers with agents. All calls are recorded and saved. Live calls can be listened in on with no disruption. All agents status are tracked and reported with accuracy within one second. Agents have the ability to mute and hold calls, conference in calls, and warm and cold transfer. All agents can receive inbound calls and dial outbound calls.
